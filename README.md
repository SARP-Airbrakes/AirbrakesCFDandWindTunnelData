# Analyses and raw data from wind tunnel testing and CFD of SARP 2025-26 Airbrakes
#This Code uses matlab to take the excel data provided in windtunnel testing to convert it into CFD data that the airbrakes can use to judge how long to deploy for to reach a traget appogee.
#In our wind tunnel we had several configurations including a dimpled nose cone/smooth nose cone, fin/no fins, 4in/5.5in scaled down airbrakes, and 0-45 degree at 5 degree intervals for the airbrakes.
#In each configuration we took data at 40,45,50 Q values to most acurately renolds number match.

#1. To use the code you have to download both the code and data from the git.
#2. You have to change the file name in the code of the downloaded data to match where you downloaded it to.
#3. You can then start to get values by changing the code at the top.
# Some options include a graph of all the cfd values for each flap angle in different configurations, graph of mulitple configurations, single values for a specific configuration.
